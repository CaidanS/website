Cellular Automata Test

Using activation functions on continuous totalistic cellular automata


A quick project to teach myself about activation functions/relearning p5.js. On each frame, the sum of each cells neighborhood is calculated, and then passed into an activation function. I used hyperbolic tangent and sigmoid, but theoretically any function is usable. I found that non-linear, strictly increasing functions provide much more interesting results. The cells are rendered as squares with HSL values mapped from their own values.

Other interesting ideas: integration with boid algorithms, using fourier transforms to approximate traditional (discrete) CA rulesets.
